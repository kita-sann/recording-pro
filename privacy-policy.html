プライバシーポリシー - Recording Pro

最終更新日: 2026年4月22日

概要

Recording Proは、画面録画・カメラ録画をGoogle Driveに保存し、AIで分析するChrome拡張機能です。本ポリシーでは、本拡張機能がどのようにデータを取り扱うかを説明します。

収集するデータ

本拡張機能は、ユーザーの個人情報を収集・送信しません。すべてのデータはユーザーのブラウザ内またはユーザーが明示的に設定したサービスにのみ保存されます。

ローカルに保存されるデータ

  • 設定情報（Google DriveフォルダID、AIプロバイダー設定、担当者リスト）
  • APIキー（OpenAI、Anthropic）- ブラウザのローカルストレージにのみ保存
  • 録画データ - 一時的にブラウザのキャッシュに保存され、保存/破棄後に削除

外部サービスとの連携

Google Drive

  • ユーザーがDrive連携を有効にした場合のみ使用
  • OAuth 2.0認証でユーザーのGoogleアカウントに接続
  • スコープ: drive.file（本拡張機能が作成したファイルのみアクセス可能）
  • 録画ファイルおよびAI分析結果をユーザー指定のフォルダにアップロード

OpenAI API

  • ユーザーがAI分析を有効にし、APIキーを設定した場合のみ使用
  • 音声データをWhisper APIに送信して文字起こし
  • 文字起こし結果をGPT-4oに送信して分析（議事録・フィードバック）

Anthropic API

  • ユーザーがAIプロバイダーとしてAnthropicを選択した場合のみ使用
  • 文字起こし結果をClaude APIに送信して分析

権限の使用目的

  • storage: 設定とAPIキーの保存
  • identity: Google OAuth2認証
  • offscreen: バックグラウンドでの録画処理
  • downloads: 録画ファイルのローカルダウンロード
  • notifications: 録画の開始・停止・エラー通知
  • alarms: 45分録画制限のタイマー

データの共有

本拡張機能は、上記の外部サービス以外にデータを送信しません。第三者へのデータ販売・共有は一切行いません。

お問い合わせ

本ポリシーに関するお問い合わせは、Chrome Web Storeのサポートページからご連絡ください。

Privacy Policy - Recording Pro

Last updated: April 22, 2026

Overview

Recording Pro is a Chrome extension that records your screen and camera, saves recordings to Google Drive, and analyzes them with AI. This policy explains how the extension handles your data.

Data Collection

This extension does not collect or transmit any personal information. All data is stored only within your browser or in services you explicitly configure.

Data Stored Locally

  • Settings (Google Drive folder ID, AI provider preferences, staff list)
  • API keys (OpenAI, Anthropic) - stored only in browser local storage
  • Recording data - temporarily cached in the browser and deleted after save/discard

External Service Integrations

Google Drive

  • Used only when the user enables Drive integration
  • Connects to the user's Google account via OAuth 2.0
  • Scope: drive.file (can only access files created by this extension)
  • Uploads recording files and AI analysis results to a user-specified folder

OpenAI API

  • Used only when the user enables AI analysis and provides an API key
  • Sends audio data to the Whisper API for transcription
  • Sends transcription results to GPT-4o for analysis (meeting minutes / feedback)

Anthropic API

  • Used only when the user selects Anthropic as their AI provider
  • Sends transcription results to the Claude API for analysis

Permission Usage

  • storage: Save settings and API keys
  • identity: Google OAuth2 authentication
  • offscreen: Background recording processing
  • downloads: Local download of recordings
  • notifications: Recording start/stop/error alerts
  • alarms: 45-minute recording limit timer

Data Sharing

This extension does not send data to any services other than those listed above. We do not sell or share your data with third parties.

Contact

For questions about this policy, please contact us through the Chrome Web Store support page.
