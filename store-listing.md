# Chrome Web Store 掲載用テキスト

## 日本語

### 短い説明（132文字以内）
画面録画・インカメラ録画をGoogle Driveに自動保存し、AIで分析する拡張機能

### 詳細説明
Recording Proは、画面録画・カメラ録画・両方同時録画に対応したChrome拡張機能です。

主な機能:
- 画面録画、カメラ録画、画面+カメラ（PiP）の3モード
- Google Driveへの自動保存
- 担当者ごとのフォルダ振り分け
- AIによる自動文字起こし（OpenAI Whisper）
- AI分析モード（議事録 / フィードバック）
- 音声レベルのリアルタイム表示
- 最大60分の録画制限と10分前の自動通知
- ローカルダウンロードにも対応

使い方:
1. 拡張機能のアイコンをクリック
2. 録画モード（画面/カメラ/両方）を選択
3. 録画開始ボタンを押す
4. 録画終了後、保存または破棄を選択

Google Drive連携:
設定画面でDrive連携をONにし、保存先フォルダIDを入力してください。録画完了後に自動でアップロードされます。

AI分析:
設定画面でAI分析をONにし、OpenAI APIキーを入力してください。録画終了後に自動で文字起こしと分析が行われます。

## English

### Short description (132 chars max)
Record screen & camera, auto-save to Google Drive, and analyze with AI

### Detailed description
Recording Pro is a Chrome extension that supports screen recording, camera recording, and simultaneous screen + camera recording.

Key features:
- Three recording modes: Screen, Camera, Screen + Camera (PiP)
- Auto-save to Google Drive
- Per-staff folder organization
- AI-powered transcription (OpenAI Whisper)
- AI analysis modes (Meeting Minutes / Feedback)
- Real-time audio level display
- 45-minute recording limit with 10-minute warning
- Local download support

How to use:
1. Click the extension icon
2. Select recording mode (Screen / Camera / Both)
3. Press the record button
4. After recording, choose to save or discard

Google Drive integration:
Enable Drive integration in settings and enter your destination folder ID. Recordings are automatically uploaded after completion.

AI analysis:
Enable AI analysis in settings and enter your OpenAI API key. Transcription and analysis are performed automatically after recording.
